Primer Paso.

Se  debe  tener  instalado  un servidor  apache 2.0 ejemplo xampp.
que  por defecto tiene  php y mysql  instalado

Segundo paso.
Se manejo  utilizo  librerias de bosstrap 5
Manejo de etiquetas Html , lenguaje php y lenguaje SQL de Mysql


Tercer paso.
se debe   crear  el  proyecto  si se esta usando xampp  en la carpeta  htdocs
del xampp

Se debe  iniciar  los servicios de php  y mysql en el xampp

Cuarto paso.

se debe   crear  la base de datos en mysql como cafeteria, luego importar el archivo cafeteria.sql y cargara las  tablas el archivo sql  esta en la carpeta bd.


Quinto paso.
ejecutar  el proyecto   en cualquier navegador de preferencia con la siguiente ruta.
http://localhost:8080/CAFETERIA/
depende de la ruta de puertos que tenga su servidor apache configurado en mi caso tengo  el puerto 8080.
